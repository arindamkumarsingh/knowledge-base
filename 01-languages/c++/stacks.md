# STACKS

Recursion calls gets stacked up like a pile of books, bucket type structure. Stacks are LIFO, add and delete both will be done from top.

If we push 10, it will be added on top. pop(delete) 10, it gets removed from the top.

QUEUE is first in first out.

### FUNCTIONS

1. **push**

2. **pop**

3. **top**(returns top value)

TIme complexity of above to be constant.

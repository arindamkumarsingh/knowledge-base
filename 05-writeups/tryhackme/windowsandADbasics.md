## File systems

In modern windows, NTFS(New Technology File System) is used, before that  FAT16/FAT32(File Allocation Table) and HFPS(High Performance File System).

FAT partitions are stiill used in USB , SD cards.

NTFS, in case of failure, it can automatically repair the files on disk using the info stored in log file before, not possible in **FAT**.

**Benefits**

* Supports file larger than 4gb.
* Specific permissions on folders and files.
* Encryption(**EFS**)
